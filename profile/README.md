## 📂 Team Projects

## 📌 1. emotion-diary

### 개요
사용자가 일기를 작성하면 AI를 통하여 내용 요악 및 감정 분석/분류 해주는 감정일기 서비스 입니다.<br/>
파이썬을 기반으로 하며, AI와 차트 등을 통해 데이터 분석, 분류 및 시각화를 학습합니다.

### 기간
- 2026.01.05 ~ 2026.02.05

### 목적
- 파이썬 문법과 FastAPI 프레임워크 학습
- AI 활용 학습
- 데이터 분석 및 시각화 학습

### 기술 스택
- Pycharm
- Python 3.12.10
- FastAPI 0.128.0
- Vue 3
